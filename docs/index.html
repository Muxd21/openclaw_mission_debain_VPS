🛠️ Core Features

VPS-Style Access

Pre-configured SSH on port 2222. Connect your PC directly with VS Code Remote-SSH for a desktop workflow.

NPM Stability

Optimized for Android limitations. Reliable flat node_modules that won't trigger PRoot symlink errors.

One-Command Sync

Stay updated with `/root/sync.sh`. Pulls the latest commits from both projects and rebuilds production assets seamlessly.

Full Hybrid Bridge

Uses `socat` on the Termux host to bridge PRoot ports. Connect via Tailscale to ports 3000, 3001, and 3011 without timeouts.

AI Powered

Includes **Perplexica**, a high-performance AI search engine, pre-installed and optimized for mobile ARM64.

Fast ARM Builds

Aggressively tracks and strips `node-llama-cpp` from pnpm monorepos to prevent 10+ minute native compilations on phone hardware.
